# Sales Performance Dashboard | Power BI

## Project Overview
Interactive sales dashboard built using Power BI 
to analyze 4-year retail performance across 
regions, product categories, and time periods.

## Tools Used
- Microsoft Excel (Data Cleaning & Preparation)
- Power BI (Dashboard & Visualization)
- DAX (Calculated Measures)

## Dataset
- Source: Kaggle — Superstore Sales Dataset
- Size: 9,994 rows | 21 columns
- Period: 2014–2017

## Key Insights
1. West region leads with highest sales share 
   (147.88K — 6.44%) among all 4 regions
2. Technology is the top-selling category 
   with 0.84M in total sales
3. Sales show consistent upward trend from 
   January to December every year
4. Hewlett Packard LaserJet 3310 Copier is 
   the most profitable product (6,983.88)
5. Total revenue of 733,215 achieved across 
   3,312 orders with 93.37K profit

## Dashboard Features
- KPI Cards with YoY trend indicators
- Interactive Year Slicer (2014–2017)
- Drillthrough pages for region-wise analysis
- Custom Tooltip pages on hover
- Cross-filtering across all visuals
- Dark theme professional design

## How to Use
1. Open .pbix file in Power BI Desktop
2. Use Year slicer to filter by year
3. Right click any region → Drillthrough
4. Hover on charts for detailed tooltips
